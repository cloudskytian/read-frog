## 欢迎来到陪读蛙博客! 🐸

我们很高兴推出陪读蛙博客,这是一个分享更新、指南和关于网页翻译及我们浏览器扩展见解的空间。

### 什么是陪读蛙?

陪读蛙是一个强大的浏览器扩展,帮助您无缝翻译网页内容。无论您是阅读文章、浏览网站还是处理文档,陪读蛙都能提供由多个 AI 提供商支持的准确翻译。

### 主要功能

#### 🌍 多提供商支持

陪读蛙支持多种翻译提供商,包括:

- **OpenAI** - GPT 驱动的翻译
- **Anthropic Claude** - 高质量的上下文翻译
- **Google Gemini** - 快速可靠的翻译
- **DeepL** - 专业级翻译
- **Ollama** - 本地化、注重隐私的翻译

#### 🎯 智能翻译模式

- **页面翻译** - 翻译整个网页,同时保留布局
- **节点翻译** - 选择并翻译特定元素
- **批量翻译** - 高效翻译多个项目

#### 🎨 自定义

- 选择您喜欢的翻译提供商
- 安全配置 API 密钥
- 自定义翻译行为
- 支持深色模式

### 我们为什么要做这个

我们注意到许多翻译工具要么在质量上妥协,要么在隐私上妥协。有些会将您的所有数据发送到外部服务器,而有些则提供不佳的翻译。陪读蛙通过以下方式弥合了这一差距:

1. **支持多个提供商** - 您可以选择最适合您的
2. **优先考虑隐私** - 您的 API 密钥保存在本地
3. **开源** - 透明且社区驱动
4. **现代化界面** - 美观、直观的界面
